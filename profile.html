About me

In 2001 my business partner and I founded Gravitate.co.nz when we launched Artfind.co.nz, a repository and service for New Zealand artists and galleries to promote themselves. Over the next 20 years we grew Gravitate into a full service web agency providing strategy, design, ux, development, implementation and hosting to a wide range of businesses and charities.

My role within Gravitate as Technical Director is primarily focused on application analysis, architecture planning, software development, infrastructure, tooling, developer support and security. I have worked with technologies such as PHP, Javascript, Go and Terraform to help solve problems for clients and our development team. I have strong experience with Linux and a deep understanding of internet technologies and adapt well to new tools and development environments and languages.

I enjoy solving problems and writing software and have a strong interest in not only developing solutions but also contributing back to community projects and tools that I use. I have contributed to the Johnny 5 javascript robotics framework, bug fixes to libraries such as RayGun and GPSD, and the Terraform provider and Go library for our hosting partner Sitehost.

Farmer Brown Golden Egg Hunt

The Farmer Brown Golden Egg Hunt was an AR experience developed in partnership with Work Communications to promote Farmer Brown eggs. The game made use of Niantic 8thwall to create an interactive egg hunt and used a combination of Open Street Map data and random points to create collectable eggs for people to find and collect. Upon finding an egg users could then switch to a camera mode to view the egg in the real world and collect a prize.

Golden Egg Hunt made use of various social login mechanisms from Facebook, Google and Apple to create a simple login flow that respected end users' privacy by only collecting user data when needed. The database was optimised for geospatial data and also allowed for detailed reporting and analysis.

Key technologies and functionality:

  • Niantic 8thWall AR framework
  • OAuth login with Apple, Facebook and Google
  • AFrame 3d javascript library
  • MySQL geospatial functionality
  • Javascript location and orientation functionality
  • Fastly CDN
  • Symfony frontend and API development
  • Terraform infrastructure provisioning and management
  • Logging and reporting with BetterStack, Raygun and SQL queries

My Humm

Myhumm is a venture between Ecrotek NZ and Awanui Scientific to provide easy to use and flexible testing of honey to apiarists, exporters, blenders and packers of honey. Myhumm is a React based portal for honey producers to place orders for honey lab tests, view their test results, create custom reports, and run blending optimisations. It provides an API that is used by Gribbles Scientific's laboratory systems to handle the placement of orders, update statuses, and collect results. It also provides a honey blending optimiser service that takes a set of test results and optimises the blending of honey to meet a set of requirements.

As the lead developer and architect on this project I was responsible for:

  • Javascript/React user interface implementation
  • Data modelling
  • Symfony/PHP backend service development
  • Implementation of provided coin-or/cbc optimization model using Python Azure Functions
  • GitHub Actions CI pipeline setup for container builds and deployment to Azure
  • Infrastructure design specification
  • Azure, Fastly CDN and DNS management using Terraform
  • PDF generation of customised lab reports for exporters

Hika

Hika was a language learning tool that was developed by a Language education specialist from the University of Auckland to assist people learning Te Reo Māori through the use of high frequency words and phrases. The first release of the application made it to the top of the app store in New Zealand.

Subsequent versions of the application also included tools for assembling the language resource packs and the ability to purchase and download new language sets from a centralised server.

It was released not long after the iPhone 3g was released and I implemented early prototypes along with the first release version. I then managed subsequent releases and updates with our design team and external mobile developers.

Responsibilities:

  • Project management
  • iOS application implementation
  • Backend service for purchasing and downloading of language packs
  • Tooling for supporting content editors and collectors
  • Supported Android development team
  • Developed cross-platform Windows Desktop and Mobile application using C# and Javascript
